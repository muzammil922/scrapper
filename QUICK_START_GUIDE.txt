STEP 1: INSTALL PYTHON
----------------------
1. Download Python 3.8+ from: https://www.python.org/downloads/
2. During installation, CHECK "Add Python to PATH"
3. Verify: Open Command Prompt, type "python --version"

STEP 2: INSTALL DEPENDENCIES
-----------------------------
Open Command Prompt in this folder, run:
   pip install -r requirements.txt

STEP 3: INSTALL CHROME BROWSER
-------------------------------
- Make sure Google Chrome is installed
- ChromeDriver will auto-install

STEP 4: RUN THE SCRAPER
------------------------
python universal_main.py

You'll see a menu:
   1. Scrape ALL categories (takes very long time!)
   2. Scrape specific categories (enter numbers)
   3. Quick test (recommended for first time)

RECOMMENDED: Start with option 3 (Quick Test)

STEP 5: CHECK OUTPUT FILES
---------------------------
After scraping completes, you'll get:
   - universal_scraped_data.csv
   - universal_scraped_data.xlsx  
   - universal_scraped_data.json

CUSTOMIZE CATEGORIES:

       "query": "search query here",
       "platforms": ["youtube", "wikipedia", "google"],
       "type": "my_type"
   }
